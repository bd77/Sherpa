version for the new SHERPA release, planned for end of February 2018

features:
- no flatweight
- RF=150
- aqi: PM10, PM25, NO2eq
- omega computed per pollutant (all cells together)
- alpha computed using 19 scenarios
- SRR produced with Python code
- for aqi PM10 and PM25 we use NOX,VOC,NH3,PPM,SO2 as precursors
- for NO2eq we use only NOx precursor
